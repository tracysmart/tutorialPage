It's one of the hardest subjects I've ever tried to learn. Some of the first concepts to learn about in the javascript language are arrays, objects, for loops, and functions. The following content is my attempt to break down each component into simple terms that are easy to digest.

Arrays

An array is a variable that holds more than one value at a time. It's like a list of items, separated by commas and enclosed within square brackets. The elements in an array are ordered and can be accessed by their index, beginning with 0 and ending at (total array size)-1. The elements of an array may include numbers, strings, booleans, objects, other arrays and more.

Objects

An object may be defined as an unordered collection of related data, in the form of “key: value” pairs. It is a list of properties within curly brackets {}. A property is a “key:value” pair, where a key is a string (also called a “property name”), and the value can be anything—a string, a number, an array, and even another object. Because an object's data is not in a strict order, one must employ different techniques than those used for an array to access the data.

For Loops

A loop is used in programming to repeat a specific block of code. The for loop provides a compact way to iterate over a range of values, such as an array. It repeatedly loops until a particular condition is satisfied.

For loops are usually structured like this:
for (statement 1; statement 2; statement 3) {
// code block to be executed
}
1. Initialization: declares the starting index, executes only once
2. Condition: defines the requirements for executing the code block
3. Increment: is executed (every time) after the code block has been executed.

The program cycles through the above commands and determines if statement 2 is true. If true, the code is executed and the loop proceeds to statement 3. There,the index is increased or decreased incrementally, and its value is checked against the condition in statement 2. The loop repeats until condition 2 is false.

Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure—a set of statements that performs a task or calculates a value. To use a function, first it must be defined.

Defining a function does not execute it, however. Defining simply names the function and specifies what to do when it is called. Calling the function actually performs the actions specified with the given parameters.

A function declaration consists of the function keyword, followed by:
1. The name of the function.
2. A list of parameters to the function, enclosed in parentheses and separated by commas.
3. The JavaScript statements that define the function, enclosed in curly brackets { }.
